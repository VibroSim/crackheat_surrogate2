crackheat_surrogate2
--------------------

This Python package is used to create the surrogate models used for
evaluating the mu and MsqrtR hidden parameters within VibroSim.

It requires R, Reticulate (interface from R to Python), and the
DiceKriging package.

R requirements (install with R install.packages()): 
  DiceKriging (CRAN)
  jsonlite (CRAN)
  reticulate (CRAN)
